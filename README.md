# Solar Flare Prediction System

## Overview
The Solar Flare Prediction System is a frontend web application designed to provide information about solar flares and their impact on space weather. The project presents a simple, responsive, and user-friendly interface built using web technologies.

## Features
- Responsive and attractive user interface
- Displays information about solar flares
- Easy navigation
- Clean and modern design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
```
solarFlare/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
